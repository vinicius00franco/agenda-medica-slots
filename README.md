# 📅 Agenda Médica - Sistema de Agendamento com Slots

Sistema moderno de agendamento médico com gestão de horários disponíveis (slots). Desenvolvido com Next.js 15, TypeScript e Tailwind CSS.

## ✨ Funcionalidades
- Cadastro e gestão de médicos
- Agendamento de consultas com slots em tempo real
- Interface responsiva e moderna
- Validações e tratamento de conflitos de horário

## 🚀 Como Rodar

```bash
# Clone o repositório
git clone https://github.com/vinicius00franco/agenda-medica-slots.git

# Entre na pasta
cd agenda-medica-slots

# Instale as dependências
npm install

# Rode o projeto
npm run dev
```

Acesse: http://localhost:3000
